== Description ==

# Change any post type author with ease.

WP Switch Author allows you to transfer massively hundreds of posts from one author to another in just
a couple of clicks. You can filter which CPT you want to transfer and the plugin will
manage everything for you.

## Use Cases when this plugin could be handy.

Below you will find some use cases when this plugin might helps
you.

1. After a light/massive __content migration__ you can use this plugin to
re-assign authors to the new imported posts.

2. After a light/massive __user import __ you can use this plugin to re-assign
posts to those new imported users.

3. If you are a developer and you need to re-assign all those pages you created to
your client admin user.

== Installation ==

1. Download the .zip plugin file clicking on the download button on this page.
2. Login in your WordPress site admin.
3. On the left hand menu, go to __Plugins -> Add new__
4. At the top of the page, click on the button __Upload Plugin__
5. Click on the button __Choose File__ and choose the .zip plugin file from your local machine.
6. After that, click on the __Install Now__ button.
7. You will be redirected to the installation page. Once the installation is finished
you will have to click on the link __Activate Plugin__.
8. Now you will find a new section on the left hand menu called __WP Switch Auhtor__. Have fun!

Alternatively, you can manually upload the plugins files via FTP or SFPT and then activate the plugin
on the Plugins section in your admin's dashboard.


== Frequently Asked Questions ==

= Would this plugin allows me to transfer default WP Post Types and Custom Post Types alike ? =

Yes, this plugin will allow you to transfer any CPT you create as long as they are public.
About WP default Post Types, at the moment only __posts, pages and Attachments__ are available to be transfered.

= What happens if I get an error message when transfering posts ? =

This plugin has been optimized to transfer the posts from one auhtor to another as fast as possible. However,
depending of the amount of posts you want to transfer and the configuration / capacity of your server, you might get
an error after the plugin transfers the posts. Usually the plugin transfers between 1000 and 1500 posts before it stops.
To continue transfering posts you only have to click the transfer button againg to continue transfering the posts until
you get a successfull notification.

= Some users are not available for me when I try to transfer the posts. Why is this happening ? =

For security reasons you cannot transfer posts to subscriber users a roles with similar capabilities.

= What if I mess up the things during a transfer ? Can I easily restore the post authors ? =

Yes, you only have to swip the authors in the list and click in the transfer button again.
